# AI Resume Analyzer

## Project Overview
An AI-powered resume analyzer that checks resume quality, ATS compatibility, keywords, and provides improvement suggestions.

## Dashboard Preview
AI Resume Analyzer dashboard showing resume score, ATS score, strengths, weaknesses, and suggestions.

## Features
- Resume upload and analysis
- ATS compatibility score
- Resume quality score
- Keyword detection
- Strength and weakness analysis
- Improvement suggestions
- Offline browser-based analysis
